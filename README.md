# University Explorer App 📱

A React Native mobile application to browse study abroad universities.

## Features
- User Authentication (Login & Signup using AsyncStorage)
- University Listing (Home Screen)
- Detailed University View
- Responsive UI (using react-native-responsive-dimensions)
- Navigation between screens

## Tech Stack
- React Native CLI
- NativeBase
- AsyncStorage
- React Navigation
